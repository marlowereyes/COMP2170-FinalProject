All clickable content should have a box shadow:
 X-Offset | 1px
Y-Offset | 5px
Blur | 10px
Spread | 3px

These shadows demonstrates that the element is able to be interacted with, such as a click. Dynamic content that is clickable should have a box shadow to create a sense of depth like a real button that can be pushed down.



All cards and buttons should have a border radius of 32px to showcase smooth round corners instead of sharp edges. This allows for a softer look that is more comfortable to users to help them remain relaxed.
